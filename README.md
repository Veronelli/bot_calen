## Bot Calen
This bot can add tasks in a Json with commands in Discord you will able:
Add a new task:
```shell
    -add [DAY] [TIME] [MESSAGE]
```
Remove a task
```shell
    -rmv [ID]
```
,or Update:
```shell
    -rvm [OPTION] [ID] [VALUE]
```
So you can see all tasks:
```shell
    -list
```
If you don't remember the commands, you can use
```shell
    -help
```
